Detailed Description

Base application. This class is a base for an OpenGL application rendering 3D objects with a perspective projection. This class provides default callbacks methods as well as the creation of a rendering context, helping to rapidly implement a simple OpenGL application using GLUT and GLEW. 

    Note that before creating any instance of this application, the static method initializeGLUT
    must be called from the main function. Moreover, the OpenGL application starts when the
    run method is called (this garantees a correct order during initializations).
Warning
This application must be a singleton (i.e. exactly one instance of Application::Base can be created in an application).
